  ## Description:
  
  * This is a command-line application that helps companies track their employee databases. It utilizes Node.js, Inquierer, and MySQL. 

  <a href="#installation"></a>
  ## Installation:
  
  * Simply type ```npm i``` to install the dependencies.

  <a href="#usage"></a>
  ## Usage:
